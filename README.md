# 学生宿舍管理系统 python52

## 项目概述

学生宿舍管理系统是一款基于Python语言开发，使用MySQL数据库进行数据存储，通过PyQt5库实现的跨平台桌面应用程序。系统致力于简化宿舍管理工作流程，提高管理效率。

## 技术栈

- **后端语言**: Python 3.7+
- **数据库**: MySQL 8.0+
- **前端框架**: PyQt5

## 功能模块

系统主要包括以下功能模块：

- 宿舍管理
- 学生信息管理
- 考勤管理
- 维修申报

具体功能模块的详细说明，请参阅后续文档。

## 系统角色

- 系统管理员：负责系统的日常维护和用户管理。
- 宿舍管理人员：负责学生宿舍的具体管理工作。

## 运行环境

- 支持操作系统：Windows、macOS

## 部署步骤

1. 准备Python 3.7+和MySQL 8.0+环境。
2. 根据部署文档，配置数据库连接。
3. 导入数据库表结构。
4. 运行PyQt5应用程序。
